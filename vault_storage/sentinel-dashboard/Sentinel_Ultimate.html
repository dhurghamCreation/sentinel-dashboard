Help & About This Application

What Is Sentinel Archive?

Sentinel Archive is a universal command center dashboard designed to give you complete visibility and control over your data vault system. It simulates an enterprise-grade shard storage monitoring platform with real-time metrics, incident management, backup controls, and more.

Key Benefits & What You Can Do

  • Monitor Live Status See capacity, latency, clusters, and threat scan rates updating in real-time every 1.5 seconds.
  • Track Integrity The Global Integrity panel shows system health with risk scoring and trend charts.
  • Manage Shards Visualize and repair a 96-shard matrix. Click any shard cell to check its status.
  • View Incident Feed Watch live operational events flow in with color-coded severity levels.
  • Analyze Resources CPU, Memory, Disk I/O, and Network throughput bars update automatically.
  • Explore the Network A geo-distributed network map shows 8 global nodes with connection lines and health status.
  • Audit Logs Search, filter, and generate sample log entries for testing and review.
  • Write & Save Notes Built-in notes pad with auto-save, export to .txt, and clipboard copy.
  • Manage Tasks Full to-do list with add/complete/delete, clear completed, and persistent storage.
  • Use the Calculator Full basic calculator with expression history and persistent memory.
  • Generate Real Reports Export system reports in PDF, CSV, JSON, or HTML format with customizable content.
  • Send API Requests Real API console with method selection, headers editor, body input, and simulated responses.
  • Run the Terminal Full terminal emulator with 12+ commands including help, status, scan, and heal.
  • Heal the System Click the Heal button to watch the autonomous repair pipeline in action.
  • Run Resilience Drills Test system response with synthetic failure simulation drills.
  • Export Snapshots Download system snapshots as JSON files for offline analysis.
  • Customize Themes 8 themes including Deep Space, Amber Glow, Nordic Frost, Aurora, and more.
  • Configure Settings Toggle auto-heal, telemetry, notifications, encryption, regions, and more.
  • Use Keyboard Shortcuts Ctrl+1-9 for page switching, Ctrl+H heal, Ctrl+E export, Ctrl+K search, Escape to close modals.
  • Data Persists Your notes, to-do items, calculator history, and selected theme are saved in your browser automatically.

Everyday Use Cases

  • Daily System Check Open the Dashboard every morning to quickly verify all systems are nominal.
  • Incident Response Watch the feed for alerts, then use the Heal button or Terminal to respond.
  • Note Taking Use the Notes page for meeting notes, system changes, or daily journaling.
  • Task Management Keep a running to-do list of maintenance tasks, upgrades, and follow-ups.
  • Quick Calculations Use the Calculator for quick data size conversions or capacity planning math.
  • Reporting Generate and export system health reports for weekly reviews or compliance audits.
  • API Testing Use the API Console to test endpoints before integrating with external tools.
  • Learning & Training New users can practice with the terminal, drills, and log generation in a safe environment.

Technical Details

  • Version 3.1.0
  • Type Single HTML file -- no server required, works offline in any modern browser
  • Storage Browser localStorage for themes, notes, todos, calculator history
  • Export JSON snapshots, TXT notes, and multi-format reports (PDF/CSV/JSON/HTML)
  • Encryption AES-256-GCM + Kyber-1024 (simulated)
